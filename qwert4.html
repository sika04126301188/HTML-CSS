- Quick reference :
In the HTML5 specification, HTML is in charge of document structure and provides a whole set of new elements for it. purpose. The specification also includes some elements with the sole task of providing styles. This is a list of those We consider most relevant: IMPORTANT: For a complete reference of the HTML elements included in the specification, visit our website and follow the links to this chapter.

- header :
This element presents introductory information and can be applied in different sections of the document. Have the purpose of containing the header of a section but can also be used to group indexes, search, logos,

- nav :
This element indicates a section of links for navigation purposes, such as menus or indexes. not all links inside a web page have to be inside a nav element, only those that are part of blocks of navigation.

- section :
This element represents a general section of the document. It is usually used to build various content blocks (for example, columns) for the purpose of ordering content that shares a specific characteristic, such as chapters or book pages, newsgroups, articles,

- aside : This element represents content that is related to the main content but is not part of it. Examples can be quotes, sidebar information, advertising.

- footer :
This element represents additional information about its parent element. For example, a footer element inserted at end of body will provide additional information about the body of the document, like the normal footer of a web page. It can be used not only for the body but also for different sections within the body, giving information information on these specific sections.

- article :
This element represents an independent piece of relevant information (for example, each article in a newspaper or each blog entry). The article element can be nested and used to display a list within another list. of related items, such as user comments on blog posts, for example.

- hgroup :
This element is used to group H elements when the header has multiple levels. (for example, a header with title and subtitle).

- figure :
This element represents an independent piece of content (for example, images, diagrams or videos) that are referenced from the main content. This is information that can be removed without affecting the fluid from the rest of the content.

- figcaption :
This element is used to display a legend or small text related to the content of a figure element, such as an image description.

- mark :
This element highlights text that is relevant in a particular situation or has been been displayed in response to user activity.

- small :
This element represents content in the margin, such as fine print (for example, disclaimers, legal restrictions, bill of rights, ).

- cite :
This element is used to display the title of a job (book, movie, poem, ).

- address :
This element encloses contact information for an article element or the document complete. It is recommended that it be inserted inside a footer element.

- time :
This element is used to display date and time in formats understandable by users and the browser. The value for users is placed between the tags while the specific for programs and browsers is included as the value of the datetime attribute. A second optional attribute called pubdate is used to indicate that the value of datetime is the publication date.
