_ quick reference :

CSS3 provides new properties to create visual and dynamic effects that are an essential part of the web these days.

- border-radius :
This property generates rounded corners for the box formed by the element. It has two different parameters that shape to the corner. The first parameter determines the horizontal curvature and the second the vertical, giving the possibility of creating an ellipsis. To declare both curve parameters, the values ​​must be separated by a slash (eg border-radius: 15px / 20px). Using only one value will determine the same form for all corners (for example, border-radius: 20px). A value for each corner can be declared in an order that go clockwise, starting at the top left corner.

-box-shadow :
This property creates shadows for the box formed by the element. It can take five parameters: the color, the horizontal offset, vertical offset, blur value, and the inset keyword to cast a shadow internal. Offsets can be negative, and the fade value and inset value are optional (for example, box-shadow: #000000 5px 5px 10px inset).

-text-shadow :
This property is similar to box-shadow but specific to text. It takes four parameters: the color, the horizontal displacement, the vertical offset, and the fade value (for example, text-shadow: #000000 5px 5px 10px).

-@font-face :
This rule allows us to load and use any font we need. First, we must declare the source, provide a name with the font-family property and specify the file with src (for example, @font-face{ font-family:Myfont; src: url('font.ttf') }). After this, we can assign the font (in the example MyFont) to any element of the document.

- linear-gradient(start position, start color, end color) :
This function can be applied to the background or background-image properties to generate a linear gradient. attributes indicate the starting point and the colors used to create the gradient. The first value can be specified in pixels, in percentage or using the keywords top, bottom, left and right. The start point can be replaced by an angle to provide a specific direction for the gradient (for example, linear- gradient(top, #FFFFFF 50%, #006699 90%);).

- radial-gradient(start position, shape, start color, end color) :
This function can be applied to the background or background-image properties to generate a radial gradient. The position of start is the origin and can be declared in pixels, percentage or as a combination of the keywords center, top, bottom, left and right. There are two values ​​for the shape: circle and ellipse, and termination points can be declared for each color indicating the position where the transition starts (for example, radial-gradient(center, circle, #FFFFFF 0%, #006699 200%);).

-rgba() :
This function is an enhancement to rgb(). It takes four values: the red color (0-255), the green color (0-255), the blue color (0-255), and opacity (a value between 0 and 1).

- hsla() :
This function is an enhancement to hsl(). It can take four values: hue (a value between 0 and 360), saturation (a percentage), lightness (a percentage), and opacity (a value between 0 and 1).

- Outline :
This property has been enhanced with the addition of another property called outline-offset. Both properties combined generate a second border away from the element's original border (for example, outline: 1px solid #000000; outline-offset: 10px;).

- border-image :
This property creates a border with a custom image. You need the border to be previously declared with the properties border or border-with, and takes at least three parameters: the URL of the image, the size of the pieces to be taken from the image to build the border, and a keyword that specifies how those pieces will be placed around the element (for example, border-image: url("file.png") 15 stretch;).

- Transform :
This property modifies the shape of an element. It uses four basic functions: scale (scale), rotate (rotate), skew (tilt), and translate (translate or move). The scale function receives only one parameter. A negative value inverts the element, values ​​between 0 and 1 shrink the element, and values ​​greater than 1 expand the element (for example, transform: scale(1.5);). The rotate function uses only one parameter expressed in degrees to rotate the element (for example, transform: rotate(20deg);). The skew function receives two values, also in degrees, for the horizontal transformation and vertical (for example, transform: skew(20deg, 20deg);). The translate function moves the object as many pixels as possible. specified by their parameters (for example, transform: translate(20px);).

- Transition :
This property can be applied to create a transition between two states of an element. Receives up to four parameters: the affected property, the time it will take to transition from start to finish, a keyword for specify how the transition will be performed (ease, linear, ease-in, ease-out, ease-in-out) and a delay value that determines the time the transition will take to start (for example, transition: color 2s linear 1s;).
