_ the new rules :

The web changed forever when a few years ago new applications built on top of Ajax implementations improved the design and user experience. Version 2.0, assigned to the web to describe a new level of development, represented a change not only in the way information was transmitted but also in how websites and new applications were designed and built.

The codes implemented in this new generation of websites soon became standard. innovation became so important to the success of any Internet project that programmers developed entire libraries to overcome limitations and meet the new requirements of designers. Lack of support by browsers was evident, but the organization responsible for web standards did not take the trends very seriously and tried to follow your own way.

Fortunately, some brilliant minds kept developing new standards in parallel and soon HTML5 was born. After the return of calm (and some agreements in between), the integration between HTML, CSS and Javascript under the tutelage of HTML5 he was like the brave and victorious knight leading the troops towards the enemy palace. Although recent turmoil, this battle started a long time ago, with the first specification of the third version of CSS.

When finally, around the year 2005, this technology was officially considered standard, CSS was ready to provide the functions required by developers (those that programmers had created years ago using Javascript codes complicated to implement and not always compatible). In this chapter we are going to study the contributions made by CSS3 to HTML5 and all the properties that simplify the lives of designers and programmers.

-css3 goes crazy :
CSS was always about styling, but not anymore. In an attempt to reduce the use of Javascript code and to standardize popular features, CSS3 not only covers web layout and styles but also shape and movement. The CSS3 specification is presented in modules that allow the technology to provide a standard specification for each aspect involved in the visual presentation of the document. From rounded corners and shadows to transformations and repositioning of the elements already presented on the screen, each possible effect previously applied using Javascript was covered.

This level of change makes CSS3 virtually unheard of technology compared to previous versions. previous. When the HTML5 specification was written with CSS in charge of design, half the battle against the rest of the proposed specifications had been won.
