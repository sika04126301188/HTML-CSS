_ New Selectors :

As we saw earlier, HTML elements have to be referenced from Javascript to be affected by the code. If you remember from previous chapters, CSS, and especially CSS3, offers a powerful referencing and selection system that is unmatched by the few methods provided by Javascript for this purpose. The getElementById, getElementsByTagName, and getElementsByClassName methods are not sufficient to contribute to the integration that this language needs and sustain the relevance that it has within the specification of HTML5. To raise Javascript to the level that the circumstances require, new alternatives had to be incorporated. from now we can select HTML elements by applying all kinds of CSS selectors via the new methods querySelector() and querySelectorAll().

- querySelector() :
This method returns the first element that matches the group of selectors specified in parentheses. Selectors are declared using quotes and the same CSS syntax, as in the following example:

function click()
{ document.querySelector(“#principal p:first-child”).onclick=showalert; }
function showalert(){ alert('clicked!'); }
window.onload=click;

Listing 4-4. Using querySelector().

In Listing 4-4, the getElementsByTagName method used earlier has been replaced with querySelector(). The selectors for this particular query are referencing the first p element that is a child of the element identified with the id attribute and the value main. Since we already explained that this method only returns the first element found, you'll probably notice that the pseudo class first-child is redundant. The querySelector() method in our example will return the first p element inside the div which is, of course, his first child. The purpose of this example is to show you that querySelector() accepts all kinds of selectors. valid CSS and now, just like CSS, Javascript also provides important tools to reference each element in the document. Multiple groups of selectors can be declared separated by commas. The querySelector() method will return the first element that matches any of them.

- querySelectorAll() :
Instead of one, the querySelectorAll() method returns all elements that match the group of selectors declared between parenthesis. The value returned is an array containing each element found in the order in which they appear in the document.

function click(){ var list=document.querySelectorAll(“#main p”);
list[0].onclick=showalert; } function showalert(){ alert('clicked!'); }
window.onload=click; Listing 4-5.

Using querySelectorAll(). The group of selectors specified in the querySelectorAll() method in Listing 4-5 will find each p element in the HTML document in Listing 4-3 that is a child of the div element. After the execution of this first line, the array list will have two values: a reference to the first element p and a reference to the second element p. Because the index of each array starts with 0, on the next line the first element found is referenced using square brackets and the value 0 (list[0]). Note that this example does not show the potential of querySelectorAll(). It will normally be used to affect several elements and not just one, as in this case. To interact with a list of elements returned by this method, we can use a loop

for: function click(){ var list=document.querySelectorAll(“#main p”);
for(var f=0; f list.length; f++){ list[f].onclick=showalert; } }
function showalert(){ alert('clicked!'); } window.onload=click;
Listing 4-6. Affecting all elements found by querySelectorAll().

In Listing 4-6, instead of selecting just the first element found, we register the onclick event handler to each of them using a for loop. Now all the p elements inside the div will display a small window when the user clicks on them. The querySelectorAll() method, like querySelector(), can contain one or more groups of selectors separated by commas. These and the other methods studied can be combined to reference elements which are difficult to reach. For example, in the next listing, the same result of the code in Listing 4-6 is achieved by using querySelectorAll() and getElementById() together:

function click(){ var list=document.getElementById('main').
querySelectorAll("p");
list[0].onclick=showalert; }
function showalert(){ alert('clicked!'); }
window.onload=click; Listing 4-7. Combining methods.

Using this technique we can see how accurate these methods can be. We can combine them on the same line and then perform a second selection with another method to reach elements within the first ones. in next chapters we will study some more examples.
